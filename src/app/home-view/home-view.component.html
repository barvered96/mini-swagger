This tool helps you document your api,

this includes creating projects with some base url

and also creating numerous resources under this project

resources such as: DELETE, GET, POST and other possibilities.

This tool is very intuitive, for project creation please select the

"Create project" option from the navigation bar.

The same goes for resource creation.

If you would like to view your projects or resources, please select

"View your projects", and for each project you may also see it's resources
